## 上個月都在幫家人做網頁

過年前這段時間，一直在用 Vibe Coding 的方式幫家人弄網頁。說起來有點不務正業，但真的很好玩。以前要搞一個完整的網頁，光前置作業就搞半天，現在整個節奏完全不同，想到什麼就能很快看到東西跑出來。

方便很多是真的，好玩也是真的。那種「腦子裡有畫面，很快就能變成真的東西」的感覺，跟以前從零開始刻完全不一樣。

## AI 這東西，我不是最近才碰

其實 AI 工具發展也不是一天兩天的事了。我之前就有在用，但說實話是斷斷續續的——有時候看到什麼新工具就玩一下，覺得「嗯不錯」，然後就放著了。過一陣子又看到新的，再玩一下，再放著。

但這一年的變化，真的跟之前不是同一個量級。

以前那些工具頂多就是幫你補補 code、回答問題，現在是真的能跟你一起做事。從「偶爾拿來用」變成「每天都在用」，這個轉變大概就是最近半年發生的。

## 從網頁版一路用到 Agent

回想一下自己的路徑還蠻有趣的：

一開始就是用網頁版的 AI，開個瀏覽器分頁，有問題就貼過去問，拿到答案再貼回來。那時候覺得已經很厲害了。

後來開始用 CLI 工具，在終端機裡直接跟 AI 互動，省掉切換視窗的麻煩。

再來就是 Agent 模式——AI 不只回答問題，而是直接幫你改檔案、跑指令、看錯誤訊息再自己修。第一次看到它自己改完 code 還會自己跑去確認有沒有 error 的時候，說沒有被嚇到是騙人的。

現在最常用的就是 **GitHub Copilot**。寫 code 的時候開著，想事情的時候問它，改東西的時候讓它幫忙。不是說它什麼都對，但那個協作的流暢度，用過就回不去了。

## 時代真的在跑

有時候會想，一年前的自己如果看到現在這個工作方式，大概會覺得很扯。但身在其中的時候，每一步好像都很自然。今天多學一個用法、明天多試一個工具，累積起來就變成完全不同的做事方式了。

快是真的快。快到有時候會覺得，不跟上的話就不是落後一點點，而是差一整個世代。

## 接下來

後面我的生活應該會有一個蠻大的轉變。細節先不說，但有一件事是確定的——我打算好好善用這些 AI 工具，把今年想做的事情一件一件推出來。

以前很多東西卡在「有想法但沒時間做」，現在工具到位了，效率跟以前完全不同。剩下的就是時間跟決心的問題。

今年會是很不一樣的一年，我有這個感覺。
